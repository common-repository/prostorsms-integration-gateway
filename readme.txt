Plugin hooks Woocommerce order and CF7 form submit to receive admin notification throught sms-service ProstorSMS

== Description ==

Plugin supports:
1. WooCommerce
2. CF7

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/smsprostor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create account with https://prostor-sms.ru/?crmplatform=wordpress
4. Use the Settings->Prostor SMS screen to configure the plugin
